重本：連絡先登録<br>

## DBの準備
### usersテーブル
<ol>
  <li>DDLフォルダにあるusers.sqlのcreate文を実行し、データベースにusersテーブルを作成する。</li>
  <li>DMLフォルダにあるusers.sqlのinsert文を実行し、テストデータを作成する。<br>  </li>
</ol>

### attendancesテーブル
<ol>
  <li>usersテーブルと同じ手順でテーブルとテストデータを作成してください。</li>
</ol>
